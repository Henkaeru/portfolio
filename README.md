# Mon Portfolio

Portfolio personnel basé sur [GitProfile](https://github.com/arifszn/gitprofile).  
Il présente mes projets, mes infos, et mon activité GitHub.

## 🔧 Tech utilisée

- GitProfile (React + GitHub API)
- Hébergement via GitHub Pages

## 📁 Personnalisation

Le contenu est géré via le fichier `gitprofile.config.js` (infos, projets, réseaux...).
